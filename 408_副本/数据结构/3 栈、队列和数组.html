栈

操作受限的线性表（只能在一端进行插入和删除）

n个元素进栈，出栈元素不同排列个数有
$$
\frac{(2n)!}{(n+1)!n!}
$$

个，被称为卡特兰数公式 ^adf7b9

队列

队头(front)，允许删除的一端
队尾(rear)，允许插入的一端

队列的顺序存储结构

判空条件q.front == q.rear == 0
不能用q.rear == MaxSize判断队满，（假溢出）

循环队列

判断队空还是队满
1 牺牲一个单元
判满：(q.rear + 1) % MaxSize == q.front
判空：q.front == q.rear
2 类型中添加size
判满：q.size == MaxSize
判空：q.size == 0
3 类型中添加tag
